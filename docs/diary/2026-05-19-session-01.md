- Welche Routes hat lokyy-app?
- Welche Backend-Endpoints existieren?
- Welche sind echt vs stub?

### Schritt 2 — Playwright-Audit
Headless durchklicken jede Sidebar-Route. Für jede:
- Lädt die Seite?
- Console-Errors?
- Zeigt sie Inhalt oder leeren-State?
- Wenn empty: gibt es einen sinnvollen Weg etwas anzulegen?

### Schritt 3 — Findings → Issues

- `audit-2026-05-19` (Sammel-Label für diese Tour)
- `priority:p0|p1|p2` (P0 = breaks first-day-user, P1 = visible-broken, P2 = nice-to-fix)
- `type:stub|missing-backend|missing-frontend|integration-broken|ux-issue`
